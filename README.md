# 🎤 Real-Time Interview Platform

A modern AI-powered interview platform that enables users to practice technical interviews in real time. The platform generates interview questions, provides an interactive interview experience, and helps users improve their communication and technical skills through AI-assisted feedback.

🌐 **Live Demo:** https://real-time-interview-platform-zeta.vercel.app/

---

## ✨ Features

- 🤖 AI-Generated Interview Questions
- 🎙️ Real-Time Mock Interview Experience
- 💬 Voice & Text-Based Interaction
- 📝 Manual Question Management
- 📊 Performance Evaluation
- 📄 Interview Summary & Feedback
- ⚡ Fast and Responsive User Interface
- 📱 Fully Responsive Design

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/praveenjaisingh/real-time-interview-platform.git
```

### Navigate to the Project

```bash
cd real-time-interview-platform
```

### Install Dependencies

Frontend

```bash
cd client
npm install
```

Backend

```bash
cd ../server
npm install
```

---

## ⚙️ Configure Environment Variables

Create a `.env` file inside the server directory.

```env
PORT=5000
GEMINI_API_KEY=YOUR_API_KEY
```

---

## ▶️ Run the Application

Start the backend:

```bash
npm run dev
```

Start the frontend:

```bash
npm run dev
```

Open your browser:

```
http://localhost:5173
```

---

## 📸 How It Works

1. Choose a topic or enter a custom interview subject.
2. Generate AI-powered interview questions.
3. Start the mock interview.
4. Answer questions using voice or text.
5. Receive AI-generated feedback and performance insights.
6. Review your interview summary to identify strengths and areas for improvement.
